Professional Experiences

05/2011 to 08/2011: Google Summer of Code

The Google Summer of Code program brings students to contribute to free software projects for a summer. I was accepted for the 2011 edition of this program by XMMS2, a project that develops a music player using the client/server architecture.

My job for the summer was to enable communication and remote procedure calls (RPC) between XMMS2 clients (before that, communication was exclusively between a client and a server). This will, in the future, provide for the development of “service” clients that implement and expose useful functionalities, thus avoiding code duplication between clients.

08/2009 to 01/2011: Librix Project

The Librix project originated from a collaboration between the OEM company Itautec and the Administration and Security Laboratory at the UNICAMP Institute of Computing, aiming to develop a GNU/Linux distribution for use in the former’s desktop computers. This distribution, which was based on Gentoo GNU/Linux, was named Librix.

During this project, I took part in the team responsible for configuring and building the releases of the distribution. I was also in charge of two parallel projects: Librix USB Creator, a tool for installing Librix to a USB device, and the development of a reduced-size Gentoo-based system for use in the Librix backup system.
