Academic Background

2014: Undergraduate Final Project

My undergraduate final project consisted of a software implementation of Lyra2, a password hashing function then under development by researchers at the University of São Paulo. The function was a finalist in the 2015 Password Hashing Competition, and my implementation is described in an article presented at the 2015 Brazilian Symposium on Information and Computational Systems Security.

2012-2013: Scientific Initiation

This research project consisted in developing an algorithm for the Steiner Cycle Problem (a generalization of the Traveling Salesman Problem) using integer linear programming.

Professional Experiences

01/2013 - present: Mozilla

I became involved with Mozilla by participating in their internship program from January to July of 2013. During that internship, I worked on Gecko, the engine that drives the Firefox browser.

Highlights of my internship include the partial implementation of a speech recognition API for the browser and important maintenance work on the Geolocation API. My internship presentation contains more details about my activities.

I have since continued working for Mozilla on the same team as a part-time contractor.

05/2014 to 08/2014: Google Summer of Code

I mentored a Google Summer of Code 2014 project at Mozilla that made great progress towards finishing the implementation of the speech recognition API I had worked on during my 2013 internship.

05/2011 to 08/2011: Google Summer of Code

I was accepted for the 2011 edition of Google Summer of Code program by XMMS2, a project that develops a music player using the client/server architecture.

My job for the summer was to enable communication and remote procedure calls (RPC) between XMMS2 clients (before that, communication took place exclusively between a client and a server). This provides for the development of “service” clients that implement and expose useful functionalities, thus avoiding code duplication between clients.

08/2009 to 01/2011: Librix Project

The Librix project originated from a collaboration between the OEM company Itautec and the Administration and Security Laboratory at the UNICAMP Institute of Computing, aiming to develop a GNU/Linux distribution for use in the former’s desktop computers. This distribution, which was based on Gentoo GNU/Linux, was named Librix.

During this project, I took part in the team responsible for configuring and building the releases of the distribution. I was also in charge of two parallel projects: Librix USB Creator, a tool for installing Librix to a USB device, and the development of a reduced-size Gentoo-based system for use in the Librix backup system.
